# MayThe4th Interactive Star Wars Lightsaber

A fun, interactive web project built with HTML, CSS, and vanilla JavaScript. This project features a highly detailed lightsaber hilt, dynamic blade ignition, and an immersive Star Wars themed background.

## Features

- **Detailed Lightsaber Hilt**: An intricately designed lightsaber hilt inspired by Luke Skywalker's *Return of the Jedi* lightsaber, created entirely using HTML and CSS gradients.
- **Dynamic Kyber Crystals**: Choose your blade color on the fly! Select from Red, Blue, Green, Purple, and Gold using the color control panel. The blade color and its glowing aura adapt instantly.
- **Immersive Space Background**: Features a pure-CSS animated twinkling starfield and a 3D-styled Death Star hovering menacingly in the background.
- **Interactive Ignition**: Click the red switch on the lightsaber's control box to ignite and retract the blade with smooth CSS transitions.

## Tech Stack

- **HTML5**: For structuring the lightsaber components and UI controls.
- **CSS3**: For advanced styling, linear/radial gradients, 3D shadows, pure-CSS animations, and CSS variables for dynamic color changing.
- **Vanilla JavaScript**: For handling click events, toggling the lightsaber blade state, and updating the CSS variables to change the blade color.

## How to Run

Running this project is incredibly simple as it requires no build tools or server!

1. Clone or download this repository.
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
4. Click the color buttons to select your blade color.
5. Click the red button on the lightsaber's control box to ignite the blade!
